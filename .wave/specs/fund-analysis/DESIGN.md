## 主流程

### 查看盈亏曲线
1. 用户进入 `/analysis`
2. 展示账户盈亏曲线图表（基于已平仓订单数据，按时间排序的累计盈亏）
3. 时间维度切换：按天/按周
4. 图表上方展示统计摘要：总盈亏、胜率、平均盈亏比

### 风险预警触发
1. 系统在后台检测连续亏损（默认 3 笔）或连续盈利（默认 5 笔）
2. 触发时在页面顶部展示黄色/红色预警横幅
3. 横幅内容：近期交易模式分析 + AI 操作建议（"建议暂停交易，冷静复盘"）
4. 用户可关闭横幅，但下次进入此页时重新展示（直到异常解除）

### 修改预警阈值
1. 页面右上角"设置"入口
2. 用户设置连续亏损触发笔数（2-10 笔）、连续盈利触发笔数（2-10 笔）
3. 保存后立即生效

---

## 分支与异常

| 场景 | 处理 |
|---|---|
| 无订单数据 | 展示空状态，引导前往"订单复盘"同步数据 |
| 未绑定 API Key | 展示绑定引导 |
| 数据不足（订单 < 3 笔） | 展示曲线但不触发预警，提示"数据不足，无法分析" |

---

## 交互状态

- 图表加载中：占位骨架
- 预警横幅：可关闭，但会话重新进入后恢复
- 统计摘要：盈利为绿色，亏损为红色
