## /alpha — Alpha 项目列表页

**布局**：顶部 Tab + 列表/表格

```
┌─────────────────────────────────────────────────┐
│  全部项目   我的关注                              │  Tabs
│─────────────────────────────────────────────────│
│  更新于 2026-06-19 06:00                        │  更新时间
│─────────────────────────────────────────────────│
│  项目       价格    7日涨跌  30日涨跌  状态   操作 │  Table Header
│─────────────────────────────────────────────────│
│  [底部盘整] TOKEN1  $0.12  -2.1%   -38.5%  ⚠️ 底部候选  [+ 关注]  │
│             TOKEN2  $1.23  +5.2%   +12.3%      —         [+ 关注]  │
│  ...                                            │
│─────────────────────────────────────────────────│
│  [加载更多] / 分页                               │
└─────────────────────────────────────────────────┘
```

**"底部候选"Badge**：橙色背景，悬停时展示 Tooltip：
```
30日跌幅：-38.5%（> 30% 阈值）
7日波动：7.3%（< 10% 阈值）
→ 符合底部盘整判断
```

## 组件需求

| 组件 | 来源 | 备注 |
|---|---|---|
| Tabs | 需新增（与 market-research 共用） | 全部/我的关注 |
| Badge | 需新增（与 market-research 共用） | 底部候选标记 |
| Tooltip | 需新增 | 底部候选判断依据 |
| Button | @trade-mind/ui | 已有，关注/取消关注 |
| Skeleton | @trade-mind/ui | 已有，加载中 |

## 可访问性

- 表格有完整 th/td 语义
- 关注/取消关注 Button 有 aria-label（"关注 TOKEN1"）
- Tooltip 键盘可触发（focus 时展示）
