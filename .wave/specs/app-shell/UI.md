# UI: app-shell — 应用骨架

> 聚合来源：auth + market-research + alpha-tracker + order-review + fund-analysis
> 状态：READY | 组件库：shadcn/ui + Tailwind CSS v4

---

## 根布局

```
┌──────────────────────────────────────────────────────┐
│  Header                                              │
│  ┌────────────────────────────────────────────────┐ │
│  │ [Logo] trade-mind   页面标题        [👤 用户头像]│ │
│  └────────────────────────────────────────────────┘ │
├────────┬─────────────────────────────────────────────┤
│        │                                             │
│  Side  │              <Page Content>                 │
│  Bar   │                                             │
│        │                                             │
└────────┴─────────────────────────────────────────────┘
```

**固定尺寸**：Sidebar 宽度 240px，Header 高度 56px

- 左：当前页面标题（动态，跟随路由）
- 右：用户头像 + Dropdown（个人资料 / 设置 / 登出）
